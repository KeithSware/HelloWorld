API Demo app illustrate the use of various Android APIs.
  - Notifications
  - Alarms
  - Progress Dialogs
  - Intents
  - Menus
  - Search
  - Persistent application state
  - Preferences
  - Background Services
  - App Widgets
  - Voice Recognition
  - And many many more...

Designed to use the compatibility library "android-support-v4.jar".
